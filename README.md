# 📸 SocialGram - Full Stack Social Media Application

A modern full-stack social media web application built using HTML, CSS, JavaScript, Node.js, Express.js, and MongoDB.

## 🚀 Live Demo

Frontend:
https://codealpha-tasks-flame.vercel.app

Backend API:
https://codealpha-tasks-apqo.onrender.com

---

## ✨ Features

- 👤 User Registration
- 🔐 User Login Authentication
- 📝 Create Posts
- ❤️ Like Posts
- 💬 Comment on Posts
- 👥 Follow Users
- 👤 Dynamic User Profile
- 📱 Responsive UI
- ☁️ MongoDB Atlas Database
- 🌐 Deployed on Vercel & Render

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/rohithg0/codealpha_tasks.git
```

### Go to the project folder

```bash
cd codealpha_tasks
```

### Install backend dependencies

```bash
cd backend
npm install
```

### Start the backend server

```bash
npm start
```

Open the frontend using Live Server or your preferred local server.
